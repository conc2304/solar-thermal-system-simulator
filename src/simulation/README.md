# Simulation Directory

Solar thermal system simulation engine with physics modeling and 3D visualization.

## Structure

- **[entities/](./entities/)** - Physical components (solar panel, tank, pump, pipes)
- **[solar-storage-system/](./solar-storage-system/)** - Main system orchestrator that connects all entities
- **[visualization-components/](./visualization-components/)** - React Three Fiber 3D visualization
- **[types.ts](./types.ts)** - Type definitions for simulation data
- **[constants.ts](./constants.ts)** - Physical constants and fluid properties
- **[utils.ts](./utils.ts)** - Helper functions for calculations and formatting

See individual READMEs for detailed documentation.
